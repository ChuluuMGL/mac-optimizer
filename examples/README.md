# Examples

This folder stores lightweight prompts and expected output shapes for testing the Mac Optimizer Skill workflow.

| Example | Purpose |
|---|---|
| `basic-maintenance/` | Diagnosis-first macOS maintenance request with dry-run and risk-ranked recommendations. |
| `basic-maintenance/sample-report.md` | Sanitized sample output for reviewing report shape and risk labels. |

The examples are intentionally synthetic. They test workflow behavior and output structure, not a specific user's Mac data.
